# 🎤 VoiceGuns (음성인식 2D 슈팅 게임)

AI를 활용한 실시간 음성인식 2인용 슈팅 게임입니다.

## 🛠 기술 스택
- **Backend:** Python (FastAPI, Socket.io)
- **Frontend:** JavaScript (Phaser.js)
- **Input:** Web Speech API

## 🚀 실행 방법
1. 서버 실행: `uvicorn server.main:combined_app --reload`
2. 클라이언트 실행: `Live Server`로 `client/index.html` 열기
